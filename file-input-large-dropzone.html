File Input with large dropzone demo

These two forms each have a single file input, which has been visually hidden. Instead, they both have a visible fake-button. The big square containers might have some JS drag-and-drop functionality added later, but these are just plain file inputs for now.

Form 1 uses a LABEL as the big square zone, and a span as the visible fake-button. Clicking anywhere inside the big square should activate the hidden file input. Also a link to BBC News acts as a way marker for keyboard navigation.

Form 2 uses a DIV as the big square zone, and a LABEL as the visible fake-button. Clicking in the big square doesn't activate the file input; you need to click the fake-button to do that. Also, another link to BBC Weather
