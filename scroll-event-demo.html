Scroll events demo

This page has a scroll event handler, which just logs the scroll event to the browser's dev tools console, using console.info(ev).

Scroll events are caused by some keypresses:

  • Up and Down keys.
  • Page Up and Page Down keys.
  • Home and End keys.
  • Pressing Space normally causes scrolling, unless a form control (e.g. a button or text input) has focus.
  • Pressing Tab may cause scrolling, if the next element in the tabbing order is outside the viewport. For example, tabbing to the links in the footer of this page.
